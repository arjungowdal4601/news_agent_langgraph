Vector Informatik has expanded its CANoe Test Package EV to include automated security testing for charging communication, targeting vulnerabilities in EV and charging infrastructure implementations at an early stage of development. The extension supports compliance with safety-critical requirements and standards including ISO 15118-2, ISO 15118-20, and Transport Layer Security (TLS) 1.2 and 1.3.

![](https://media.automotiveworld.com/app/uploads/2026/04/14105838/vector-canoe-test-package-ev-security-graphic-1024x655.jpg)


The addition addresses limitations of manual security testing, which Vector describes as time-consuming, difficult to reproduce and poorly suited to integration into continuous integration and delivery (CI/CD) pipelines. The automated approach enables reproducible validation of charging communication and is designed to slot into existing development workflows, with systematic result documentation reducing manual overhead throughout the development cycle.

Functional coverage includes vehicle-to-grid (V2G) communication fuzzing with system-under-test monitoring, TLS fuzzing, TLS protocol tests and custom security tests through defined interfaces. Plug & Charge functionality is also within scope. The package will be updated continuously with additional functions over time.
